# BACNB Tasks

This folder contains the four individual OpenSesame/OSWeb tasks used by the reduced BACNB battery.

Each task folder includes:

- an OpenSesame experiment file (`.osexp`);
- an OSWeb HTML export (`index.html`).

No raw participant data or generated analysis outputs are included.

## Language Status

The current task interfaces are in Brazilian Portuguese (PTBR). They are provided as the original working BACNB task versions. The scientific and developer communities are invited to create more polished versions, translations, localization files, and language-specific adaptations, provided that attribution is preserved and the repository license terms are followed.
