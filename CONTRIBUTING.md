# Contributing to BACNB

BACNB is shared to support open science, reproducibility, user freedom, and public benefit.

Contributions are welcome, including:

- translations and localization of the PTBR task interfaces;
- more polished OpenSesame/OSWeb task versions;
- improvements to task presentation, usability, and accessibility;
- code improvements to the Monte Carlo simulation;
- documentation fixes and methodological clarification.

The current SART, SST, Flanker, and Digit Span task interfaces are in Brazilian Portuguese (PTBR). This is intentional: these are the original working task versions. Community translations and cleaner localized versions are encouraged.

All contributions must preserve appropriate attribution and follow the repository license terms:

- source code and computational algorithms: GNU GPLv3;
- scientific documentation and original task-design material: CC BY 4.0;
- third-party OpenSesame/OSWeb components: their respective upstream licenses.

By contributing to this repository, contributors agree that their contributions are provided under the applicable project license for the file or material being changed. Contributors retain authorship of their own contributions unless a separate written agreement says otherwise.

The author does not seek royalties from community use. The goal is scientific usefulness, independent verification, and benefit to the public.
